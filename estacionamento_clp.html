03 / Rung 1
Rung 1 — Entrada de veículos
O portão de entrada só abre se o sensor detectar um carro E o estacionamento não estiver lotado.

1
Arraste ⊣⊢ (1º ícone amarelo) para o rung → clique nele → selecione sensorEntrada
2
Arraste ⊣/⊢ (2º ícone, tem barra) ao lado direito → clique → selecione sistemaLotado
3
Arraste (O) bobina no final → clique → selecione portaoEntrada
4
Arraste 006 contador → clique → selecione contadorEntrada
⊣⊢ NA = passa quando TRUE (carro detectado). ⊣/⊢ NF = passa quando FALSE (não lotado). Os dois em série = porta E.

05 / Rungs 3–6
Rungs 3 ao 6 — Lógica de controle
Calcula vagas ocupadas, ativa o bloqueio quando lotado e acende a luz de aviso.
Rungs 3 ao 6
// Rung 3: Subtração — vagas ocupadas
vagasOcupadas = entradas − saídas
SUB
IN1: contadorEntrada.CV
IN2: contadorSaida.CV
OUT: vagasOcupadas
// Rung 4: SET lotado quando vagas ≥ 10
vagasOcupadas ≥ totalVagas
GEQ ≥
IN1: vagasOcupadas
IN2: totalVagas
sistemaLotado
SET
// Rung 5: RESET lotado quando vagas < 10
vagasOcupadas < totalVagas
LES <
IN1: vagasOcupadas
IN2: totalVagas
sistemaLotado
RESET
// Rung 6: Acende luz quando lotado
sistemaLotado
luzLotado
R3
Bloco [+×] SUB → IN1=contadorEntrada.CV, IN2=contadorSaida.CV, OUT=vagasOcupadas
R4
Bloco [≥] GEQ → IN1=vagasOcupadas, IN2=totalVagas + bobina SET sistemaLotado
R5
Bloco [<] LES → IN1=vagasOcupadas, IN2=totalVagas + bobina RESET sistemaLotado
R6
Contato ⊣⊢ sistemaLotado + bobina (O) luzLotado
